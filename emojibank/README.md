An educational Progressive Web App that teaches children about compound interest through a fun, emoji-themed piggy bank tracker.

## Features

- **Two Piggy Banks**: Separate tracking for two children
- **Automatic Weekly Interest**: Compound interest calculated automatically
- **Visual Growth Charts**: See savings grow over time
- **Configurable Interest Rate**: Adjust learning pace
- **Mobile-Friendly**: Works on phones, tablets, and desktops
- **Offline Capable**: PWA features for offline use

## How It Works

1. **Deposit Money**: Add allowance, gift money, or chore earnings
2. **Watch It Grow**: Interest accrues automatically every week
3. **Visual Learning**: Charts show compound growth over time
4. **No Withdrawals**: Encourages saving by not allowing spending

## Educational Value

- Teaches compound interest concepts
- Reinforces delayed gratification
- Builds financial awareness
- Age-appropriate for K-2 (ages 5-7)

## Technical Stack

- Vanilla JavaScript (ES6+)
- Vanilla CSS with Grid/Flexbox
- localStorage for persistence
- No build process - pure HTML/CSS/JS

## Usage

Simply open `index.html` in a web browser. No installation or build steps required.

For local development:
```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
